# Artificial Neural Network (ANN) Project

A comprehensive project focused on preprocessing data, experimenting with architectures, and training an Artificial Neural Network model. 

---

## 📂 Repository Structure

The project is organized as follows:

*   **`Artificial_Neural_Network-main/`** - The root directory containing the project source.
    *   **`Dataset/`** - Contains the raw and structured data files used for training and testing.
    *   **`Train Model/`** - Includes scripts/notebooks dedicated to definition, compilation, and training of the ANN.
    *   **`Experiment/`** - Notebooks or scripts tracking hyperparameter tuning and model architecture tests.
    *   **`AI_Assisted/`** - Code, logs, or assets related to AI-generated or assisted optimizations.
    *   **`Preprocessed.ipynb`** - Jupyter notebook handling data cleaning, encoding, scaling, and feature engineering.

---

## 🚀 Getting Started

### 1. Prerequisites
Ensure you have Python installed along with the required machine learning libraries. You can set up your environment using jupyter.
